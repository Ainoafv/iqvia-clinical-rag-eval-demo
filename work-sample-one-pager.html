What it demonstrates

  • Grounded RAG — clinical questions answered with mandatory source citations from fictional protocol corpus
  • Claim-level grounding check — supported / unsupported / needs-review; automatic block on hallucinated numeric data
  • Evaluation harness — hit@k, groundedness %, golden test suite (5/5 pass)
  • Structured agent output — JSON schema visible (dose, route, confidence, hallucination flags)
  • Never / Ask / Auto policy ladder — code-enforced; NEVER blocks auto-publish to EHR
  • Human-in-the-loop — approve, reject, or edit before any answer leaves the queue
  • Append-only audit log — exportable trace of every pipeline step and human decision

Design principle

"I treat AI outputs as hypotheses to validate, not results to trust." The model proposes; the evaluation gate and a qualified reviewer decide.

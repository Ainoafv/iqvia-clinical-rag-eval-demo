What it demonstrates

  • Grounded RAG — clinical questions with mandatory source citations from fictional protocol corpus
  • Claim-level grounding — supported / unsupported; automatic block on hallucinated numeric data (Q2 eval trap)
  • Evaluation harness — hit@k, groundedness % vs 80% gate, golden regression suite (5/5)
  • Structured output — Pydantic JSON schema (confidence, hallucination flags)
  • Never / Ask / Auto policy ladder — code-enforced release permissions
  • Append-only audit log — exportable trace of every pipeline step

Design principle

"I treat AI outputs as hypotheses to validate, not results to trust." The model proposes; the evaluation harness cross-checks and gates release.
